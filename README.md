# Project Overview
Developed an interactive Excel dashboard to analyze Vrinda Store’s sales data and customer behavior. The project focuses on converting raw data into meaningful insights to support business decision-making.

# objective 
* To analyze sales performance across different categories and channels
* To identify customer trends and buying patterns

# Tools Used
* Microsoft Excel
* Pivot Tables & Pivot Charts
* Excel Functions
* Slicers & Data Visualization

# Steps Covered
* Data Cleaning (removed duplicates, handled missing values)
* Data Transformation (formatted and structured dataset)
* Data Analysis (KPIs like sales, orders, categories)
* Dashboard Creation (interactive visuals using charts & slicers)

# Sample Insights
* Adult age group (30-49 yrs) is max contributing (50%)
* Amazon,Flipkart and Myntra channels are max contributing (80%)
* Men are less buy to compared to women (65%)
* Maharashtra,karnataka and uttar pradesh are max contributing (80%)
